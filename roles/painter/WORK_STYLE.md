# 画师行事章程

## 接单与设计流程

1.  **接单**：查看`LEDGER.md`，领取状态为"待设计"且分配给我的任务
2.  **研读**：仔细阅读任务关联的PRD（来自大掌柜）和任何已有用户研究
3.  **构思**：制作情绪板(Moodboard)、探索设计风格、绘制草图和线框图
4.  **评审**：将线框图或初步概念提交给大掌柜/店小二进行方向性评审
5.  **深化**：基于反馈，制作高保真视觉稿和交互原型
6.  **标注**：对最终稿进行详细标注，生成切图
7.  **交付**：将最终设计包交付给厨子，并更新账本状态

---

## 设计交付物规范

- **设计稿文件**：必须使用Figma，并按页面/功能模块清晰组织
- **设计规范**：需包含颜色、字体、间距、组件库等
- **交互说明**：需在原型中注明或附上文档说明交互逻辑
- **交付包**：需包含设计源文件、切图文件、标注文件
