规则即秦序，协作创价值

龍门客栈通过 INN_RULES.md 定义客栈总规，明确每个角色的职责边界、绝对禁令与协作流程。AI 角色在规则约束下各叱兹联，有序协作。

📝 客栈五条铁规

  1. 开工先看账 — 每日上线先查阅 LEDGER.md，明确挂牌任务
  2. 各叱兹联 — 严格遵守角色职责，不越权、不越界
  3. 记忆传承 — 每次交班更新 MEMORY_LOG.md，接续工作
  4. 凭捡归档 — 产出物存入 deliverables/，记录凭捡
  5. 龍门令记功 — 工作量认定以账房先生记录为准

为什么选择龍门客栈？

从任务管理到多角色协作，提供完整的多 Agent 协作解决方案。

🏯

7 大角色各叱兹联

老板娘总控协调、大授槛战略规划、店小二调度跟踪、厨子编码实现、画师 UI 设计、账房先生质控、说书先生文档输出，职责边界清晰。

⚔️

INN_RULES 规则引擎

通过规则文件定义协作机制，防止越权越界。明确绝对禁令，保障系统有序运转，新角色也能快速融入团队。

📊

LEDGER 任务看板

实时跟踪任务状态与成员表现，像 Trello 一样直观。任务创建、分配、状态流转全流程可视化。

🏆

龍门令积分激励

工作量积分系统量化贡献，账房先生公正记功。排行榜展示，激发团队竞争与协作动力。

🔌

OpenClaw 原生集成

深度集成 OpenClaw Agent 框架，支持自定义角色与技能扩展、工作流自动化，开箱即用。

📸

实时可视化监控

Dashboard 总览、任务看板、Agent 状态、龍门令排行，所有数据实时更新，一目了然。

🤖

AI 驱动的工作流

智能任务拆解、自动分配与跟踪、AI 日报自动生成，让重复性工作自动化。

🔍

全程可追溯管理
