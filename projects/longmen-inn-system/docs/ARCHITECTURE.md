## 技术选型说明

### 后端技术栈

| 技术 | 版本 | 选型理由 |
|------|------|----------|
| **Python** | 3.11+ | 语法简洁，生态丰富，适合快速开发 |
| **FastAPI** | 0.104+ | 高性能异步框架，自动生成API文档，类型提示支持 |
| **SQLAlchemy** | 2.0+ | ORM行业标准，支持异步，强大的查询构建器 |
| **PostgreSQL** | 15+ | 功能强大的开源关系数据库，支持JSON、全文检索 |
| **Redis** | 7+ | 高性能缓存和消息队列，支持多种数据结构 |
| **Celery** | 5.3+ | 分布式任务队列，支持定时任务、重试机制 |
| **Pydantic** | 2.5+ | 数据验证和序列化，与FastAPI深度集成 |

### 前端技术栈

| 技术 | 版本 | 选型理由 |
|------|------|----------|
| **TypeScript** | 5.2+ | 静态类型检查，提高代码质量和可维护性 |
| **React** | 18.2+ | 组件化开发，丰富的生态系统 |
| **Vite** | 5.0+ | 极速的构建工具，支持热更新 |
| **Ant Design** | 5.12+ | 企业级UI组件库，设计规范完善 |
| **Zustand** | 4.4+ | 轻量级状态管理，简单易用 |
| **Axios** | 1.6+ | HTTP客户端，支持拦截器和请求取消 |
| **ECharts** | 5.4+ | 数据可视化图表库 |

### 基础设施

| 技术 | 用途 | 选型理由 |
|------|------|----------|
| **Docker** | 容器化 | 环境一致性，便于部署 |
| **Docker Compose** | 本地编排 | 快速搭建开发环境 |
| **Nginx** | 反向代理 | 高性能，支持负载均衡 |
| **GitHub Actions** | CI/CD | 与GitHub深度集成 |
